# **App Name**: EcoVision

## Core Features:

- Waste Classification: Use the device's camera to classify waste into Plastic, Paper, and Organic categories using a Teachable Machine model.
- Camera Integration: Integrate with the device's camera to capture images for classification.
- Theme Toggle: Allow users to switch between a light and dark theme.
- Classification History: Maintain a closable sidebar (toggle-controlled) displaying previously scanned waste types.

## Style Guidelines:

- Primary color: Forest green (#38A3A5) to evoke environmental consciousness.
- Background color: Very light, desaturated green (#F0F8F0) to provide a calm and natural backdrop.
- Accent color: Earthy brown (#A36F38) to complement the green and provide a grounded feel.
- Font pairing: 'Inter' sans-serif for body text and 'Space Grotesk' sans-serif for headlines, creating a modern yet readable interface.
- Use simple, clear icons to represent waste categories in the sidebar.
- Sidebar is positioned on the right and is closable with a toggle button without interfering with the main classification workflow.
- Subtle transition animations for theme toggle and sidebar open/close.
